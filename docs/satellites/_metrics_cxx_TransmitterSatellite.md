### Data Transmission

| Metric | Description | Value Type | Interval |
|--------|-------------|------------|----------|
| `TX_BYTES` | Amount of bytes transmitted during current run | Integer | 5s |
| `TX_BLOCKS` | Number of data blocks transmitted during current run | Integer | 5s |
| `TX_RECORDS` | Number of data records transmitted during current run | Integer | 5s |
