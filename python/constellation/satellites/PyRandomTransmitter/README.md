## Parameters

| Parameter | Type | Description | Default Value |
|-----------|------|-------------|---------------|
| `block_size` | Unsigned integer | Size of a data block in bytes | `1024` |
| `number_of_blocks` | Unsigned integer | Number of data blocks per data message | `1` |
